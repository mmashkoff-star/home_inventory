# home_inventory
Простое приложение для домашней инвентаризации: помогает вести список вещей с указанием места хранения (комната, шкаф, полка) и быстро находить, где что лежит.
Этот проект — лёгкий инструмент для учёта домашних вещей: одежды, техники, посуды, инструментов и любых других предметов дома. Приложение позволяет добавлять вещи с названием, категорией и точным местом хранения, а затем искать их по названию или категории, чтобы быстро вспомнить, где что лежит. Цель проекта — навести порядок в доме и избавить от ситуации «знаю, что это у меня есть, но не помню, куда положил».
